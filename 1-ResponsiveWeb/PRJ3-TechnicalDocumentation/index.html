Git & GitHub

Mục tiêu: Nắm vững các lệnh dòng lệnh (CLI) và cách sử dụng Git để quản lý mã nguồn hiệu quả trên GitHub.

1. Giới thiệu về Git

Git là một hệ thống quản lý phiên bản phân tán giúp theo dõi các thay đổi của mã nguồn trong quá trình phát triển phần mềm.

2. Các lệnh Git cơ bản

  • git init - Khởi tạo kho lưu trữ Git mới.
  • git clone [URL] - Sao chép một repository từ GitHub.
  • git status - Kiểm tra trạng thái của các tệp.
  • git add [file] - Thêm tệp vào vùng staging.
  • git commit -m "Thông điệp commit" - Lưu thay đổi với mô tả.
  • git push - Đẩy mã nguồn lên GitHub.
  • git pull - Lấy mã nguồn mới nhất từ GitHub.

3. Làm việc với GitHub

GitHub là nền tảng lưu trữ mã nguồn phổ biến sử dụng Git để làm việc nhóm.

Bước 1: Tạo repository

Truy cập GitHub, đăng nhập và nhấn "New Repository".

Bước 2: Kết nối Git với GitHub

git remote add origin [URL repository]
git branch -M main
git push -u origin main

Bước 3: Làm việc nhóm với GitHub

Sử dụng Pull Request, Issues, Code Review để cộng tác hiệu quả.

4. Kết luận

Git & GitHub giúp quản lý mã nguồn, theo dõi thay đổi và làm việc nhóm dễ dàng.
